%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%

m && \textbf{M}ama \\ % no-dictionary

%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%
\subsection*{Groß- und Kleinschreibung}
%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%
%
Alle offiziellen Wörter werden immer, selbst am Satzanfang, klein geschrieben. 
Großbuchstaben werden nur für inoffizielle Wörter benutzt. 
Dies sind z.B. Eigennamen, Länder, Sprachen und Religionen. 
%
%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%
\subsection*{Satzzeichen}
%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%
%
Achte bitte immer auf korrekte Satzzeichen.
Falsche beziehungsweise fehlende Satzzeichen beeinträchtigen die Verständlichkeit.

\begin{supertabular}{p{2,5cm}|ll}
\textbf{.} && \textit{Separator}: Beendet wird ein Aussagesatz mit einem Punkt. \\ % no-dictionary
\textbf{!} && \textit{Separator}: Die Befehlsform und Interjektionen enden mit einem Ausrufungszeichen. \\ % no-dictionary
\textbf{?} && \textit{Separator}: Beendet wird eine Frage mit einem Fragezeichen. \\ % no-dictionary
\textbf{:} && \textit{Separator}: Ein Doppelpunkt trennt zwei (Halb)-Sätze. \\  % no-dictionary
\textbf{,} && \textit{Separator}: Ein Komma wird nach \glqq o\grqq verwendet wenn man Leute anspricht. \\ && Optional kann es vor einer Präposition eingefügt werden. \\ % no-dictionary
\end{supertabular} \\
